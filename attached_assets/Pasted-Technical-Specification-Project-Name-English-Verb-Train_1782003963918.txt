Technical Specification
Project Name

English Verb Trainer

A Progressive Web App (PWA) inspired by Conjugato for Spanish verbs, but focused on English verb forms, tenses, and usage in context.

The application is designed primarily for self-study and deliberate practice of English verb forms.

Main Goals

The app should:

work offline after installation
be installable on iPhone and Android as a PWA
have a clean, minimalist UI
prioritize speed and repetition
require minimal navigation
focus on active recall rather than passive review

The user should be able to answer hundreds of exercises in a short session.

Technology Requirements
React
TypeScript
Vite
PWA support
IndexedDB for local storage
No backend required
Entire app works locally
Mobile-first design
Learning Modes

The app should support four exercise modes.

Mode 1: Verb Form Production

The app displays:

Verb: write

Tense: Past Perfect

Subject: we

The user types:

we had written

or

had written

(accept both options depending on settings)

Examples

Verb: go

Tense: Present Perfect

Subject: she

Answer:

she has gone

Verb: work

Tense: Future Simple

Subject: they

Answer:

they will work

Mode 2: Tense Recognition

The app displays a complete sentence.

Example:

She has been working here since 2020.

The user must identify:

Present Perfect Continuous

Example:

The letter had been written before I arrived.

Answer:

Past Perfect Passive

Goal

Train recognition of English tense structures.

Mode 4: Context Gap Fill

The app displays a sentence.

Example:

Yesterday I _____ to the store.

(go)

Answer:

went

Example:

By the time she arrived, we _____ dinner.

(finish)

Answer:

had finished

Example:

This house _____ in 1905.

(build)

Answer:

was built

Goal

Practice grammar in context rather than isolated tables.
